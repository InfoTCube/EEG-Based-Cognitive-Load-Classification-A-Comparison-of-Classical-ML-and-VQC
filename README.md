# EEG-Based Cognitive Load Classification: Classical ML vs Variational Quantum Classifier

Binary classification of cognitive load from EEG signals, comparing four classical machine learning models (Logistic Regression, SVM, Random Forest, XGBoost) with a Variational Quantum Classifier (VQC) built with PennyLane.

---

## Motivation

Cognitive overload affects an estimated 15–20% of drivers annually and contributes to roughly 1 million accidents per year worldwide. Current detection methods — subjective questionnaires, eye-tracking, manual EEG analysis — are either unreliable, expensive, or too slow for real-time use. Automated EEG classification could enable real-time detection in vehicles, classrooms, and operating rooms.

This project establishes a realistic performance baseline for cross-subject EEG classification and evaluates whether near-term quantum machine learning provides any measurable advantage over classical methods.

---

## Dataset

**EEGMAT** — *EEG During Mental Arithmetic Tasks* (Zyma et al., 2019)  
Source: [PhysioNet](https://physionet.org/content/eegmat/1.0.0/)

| Property | Value |
|----------|-------|

## Pipeline

### 1. Data Loading

`src/data_loader.py` reads all `Subject##_[1|2].edf` files with MNE, applies a 1–50 Hz bandpass filter, and returns a list of recordings in µV.

### 2. Preprocessing

`src/preprocessing.py` slices each continuous recording into 2-second non-overlapping epochs and performs a **subject-aware train/test split** — all epochs from a given subject go exclusively to train or test, preventing data leakage and giving a realistic cross-subject generalisation estimate.

| Split | Subjects | Epochs |
|-------|----------|--------|
| Train | 28 | 3,413 |
| Test | 8 | 925 |

### 3. Feature Engineering

`src/features.py` extracts 5 neuroscientifically motivated spectral features per epoch using Welch's method:

| # | Feature | Region | Band | Cognitive basis |
|---|---------|--------|------|-----------------|
| 1 | Theta/Beta Ratio | Frontal | 4–8 Hz / 12–30 Hz | Canonical cognitive load index; amplifies condition differences |
| 2 | Frontal Theta Power | Frontal | 4–8 Hz | Working memory and executive control drive frontal midline theta |
| 3 | Central Beta Power | Central | 12–30 Hz | Increases with active cognitive processing and task engagement |
| 4 | Frontal Beta2 Power | Frontal | 15–22 Hz | More specific to executive function than broadband beta |
| 5 | Spectral Entropy | Frontal + Central | Full spectrum | Measures signal complexity; modulated by neural coordination under load |

All features are statistically significant (Mann-Whitney U, p < 0.05) with effect sizes ranging from |d| = 0.02 (Spectral Entropy) to |d| = 0.30 (Theta/Beta Ratio).

### 4. Classical ML

Four classifiers trained on `StandardScaler`-normalised features, all compensating for the ~3:1 class imbalance:

- **Logistic Regression** — linear baseline with L2 regularisation
- **SVM (RBF kernel)** — non-linear, effective with small feature sets
- **Random Forest** — bagging ensemble with built-in feature importance
- **XGBoost** — gradient boosting with `scale_pos_weight`

Evaluated with 5-fold stratified cross-validation on training data (no test leakage) and held-out test set metrics.

### 5. Variational Quantum Classifier

A data re-uploading VQC implemented with PennyLane on a classical state-vector simulator:

| Component | Detail |
|-----------|--------|
| Qubits | 5 (one per spectral feature) |
| Feature map | `AngleEmbedding (Ry)` re-uploaded at every layer |
| Ansatz | `StronglyEntanglingLayers`, 3 layers |
| Parameters | 45 trainable weights |
| Loss | Unweighted MSE on {−1, +1} labels |
| Imbalance handling | Stratified mini-batches (16 per class) |
| Optimiser | Adam, lr = 0.015, 400 steps |
| Training time | ~9 minutes (CPU) |

Data re-uploading (Perez-Salinas et al., 2020) makes the VQC a universal approximator by re-encoding inputs at each variational layer, analogous to a deep neural network that reuses its input at every layer.

## Results

All models trained on the same 3,413-epoch training set and evaluated on the same 925-epoch test set (8 unseen subjects). **F1 and AUC-ROC are the primary metrics** — raw accuracy is misleading because a trivial "always predict Rest" baseline scores ~73%.

### Epoch-Level Metrics

| Model | Accuracy | F1 | AUC-ROC | Precision | Recall |
|-------|----------|----|---------|-----------|--------|
| **Random Forest** | 0.665 | **0.520** | **0.711** | 0.418 | 0.690 |
| SVM (RBF) | 0.685 | 0.496 | 0.698 | 0.420 | 0.609 |
| XGBoost | 0.694 | 0.426 | 0.653 | 0.477 | 0.387 |
| VQC | 0.619 | 0.447 | 0.623 | 0.366 | 0.573 |
| Logistic Regression | 0.397 | 0.380 | 0.522 | 0.261 | 0.698 |

### Recording-Level Accuracy (Majority Vote)

Aggregating epoch predictions by majority vote per subject/session — one decision per recording — mirrors real deployment and substantially boosts accuracy.

| Model | Recording Accuracy | Recording F1 |
|-------|-------------------|--------------|
| **Random Forest** | **0.812** | **0.762** |
| SVM (RBF) | 0.750 | 0.667 |
| VQC | 0.688 | 0.667 |
| XGBoost | 0.688 | 0.615 |
| Logistic Regression | 0.562 | 0.462 |

### Key Findings

- **Random Forest wins overall**: best F1 (0.52) and AUC (0.71) at epoch level; majority voting pushes recording-level accuracy to 81.2%.
- **VQC beats random chance** (AUC 0.623 vs 0.500 baseline) and recalls >57% of mental-math epochs, demonstrating that a hybrid quantum-classical circuit can genuinely learn from real biosignal data.
- **VQC cannot match classical models** on this dataset: 45 parameters vs. millions of effective parameters in ensemble methods; high inter-subject variability requires memorisation capacity the VQC lacks.
- **XGBoost has deceptively high accuracy** (0.694) but the lowest Mental Math recall (0.387) — it over-predicts Rest.

---

## Why Classical ML Wins Here

1. **Capacity mismatch** — RF and XGBoost have orders-of-magnitude more effective parameters than a 45-param VQC on a 3,400-sample dataset.
2. **Barren plateaus** — gradient magnitudes shrink with VQC circuit depth, slowing convergence relative to classical backprop.
3. **Feature geometry** — hand-crafted spectral features form axis-aligned clusters that decision-tree splits exploit directly.
4. **Inter-subject variability** — 8 test subjects differ substantially in spectral profiles; the VQC cannot memorise per-subject offsets.
